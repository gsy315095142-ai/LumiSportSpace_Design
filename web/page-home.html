🏟️ 首页：赛事选择

选择门店后进入的赛事主页 — 三种角色、两个 Tab、完整赛事生命周期

📖页面定位

「首页：赛事选择」是用户选择门店后进入的核心阶段。在此阶段，管理员负责配置玩法项目与模式，选手等待报名开放，观众等待竞猜开放。

管理员通过赛事主页导航栏的「管理」按钮进入独立的场次管理页面（admin.vue），在此完成玩法选择、确认玩法、开放报名等操作。选手和观众则留在赛事主页（event.vue）等待状态推进。

🛡️管理员：场次管理页面

管理员点击赛事主页导航栏的「管理」按钮后，进入独立的场次管理页面。页面结构随当前阶段动态变化：

🛡️ 场次管理页面（玩法配置阶段）

管理员 玩法配置阶段的界面

  • 场次流程进度条 — 顶部显示 6 步流程：玩法配置 → 等待报名 → 报名中 → 竞猜中 → 比赛中 → 已结算。当前步骤高亮显示，下方提示当前阶段操作指引
  • 场次名称输入框 — 可自定义赛事名称，如「春季友谊赛」
  • 玩法项目切换按钮 — 三个按钮：冰球 / 拳王 / 击剑。点击切换时有二次确认弹窗
  • 冰球大师模式开关 — 仅当玩法选为「冰球」时显示。开启后结算时由 Server 自动上报冰/火/风球出现次数
  • 「确认玩法」按钮 — 填写完玩法配置后点击此按钮，系统创建首局场次并进入「等待报名」阶段。此时 Unity Server 端会自动跳转进入对应玩法的匹配页面，等待选手就位
  • 确认后界面变化 — 点击「确认玩法」后：配置字段自动锁定为只读；出现顶部 Tab 栏（场次管理 / 竞猜大盘池）；流程操作区出现「重选玩法」和「开始报名」两个按钮

⚠️ 配置锁定规则

场次名称、玩法项目、冰球大师模式在点击「确认玩法」后自动锁定，进入等待报名阶段后不可修改。如需修改，必须通过「重选玩法」退回到玩法配置阶段。进入报名阶段后，「重选玩法」按钮消失，只能通过「重新等待报名」退回。

📑赛事主页子 Tab 详解

选手和观众在赛事主页底部看到的两个子 Tab：

🎰 我要竞猜

竞猜操作与排行榜

  • 信息条：昵称 + 金币余额
  • 胜负盘 — 押红/蓝方获胜（所有游戏）
  • 精准总分 — 猜最终总得分（所有游戏）
  • 精准分差 — 猜红蓝分差（所有游戏）
  • 元素之王 — 猜冰/火/风球最多者（仅冰球大师模式）
  • 躺平之王 — 猜双方倒地总次数（仅拳王）
  • 竞猜收益排行榜 → 更多跳转

⚠️ 需管理员开启竞猜后才能下注；已参赛选手不可下胜负盘

🤺 我要参赛

报名操作与排行榜

  • 信息条：昵称 + 三玩法积分
  • 状态提示 — 当前能否操作的文案
  • 选边 — 蓝/红阵营二选一
  • 提交报名 — 进入待审核状态
  • 撤销报名 — 取消已提交的报名
  • 排行榜（可切换玩法查看）

⚠️ 仅报名阶段可操作；提交后需管理员审核通过才成为正式选手

🤺选手：赛事主页（等待报名开放）

选手留在赛事主页，等待管理员确认玩法并开放报名。此时无法进行任何操作：

🤺 选手视角（等待阶段）

选手 等待报名开放时

  • 导航栏无「管理」按钮 — 普通用户（user 角色）看不到管理入口
  • 本场对阵模块显示「新的赛事即将开始……」 — 尚无对阵信息，等待管理员操作
  • 信息条显示个人积分 — 展示昵称 + 三种玩法的积分（默认 100 分）
  • 「我要参赛」Tab 显示等待提示 — 「新一轮报名尚未开始（待管理员开放报名）」，蓝/红选边按钮灰色不可操作
  • 排行榜可浏览 — 即使无法参赛，仍可查看各玩法的积分排名

👥观众：赛事主页（无法押注）

观众同样留在赛事主页，但切换到「我要竞猜」Tab 查看竞猜面板。此时竞猜尚未开启，无法进行押注：

👥 观众视角（等待阶段）

观众 无法押注时

  • 「我要竞猜」Tab 显示竞猜面板 — 但由于管理员尚未开启竞猜，所有玩法卡片显示不可下注状态
  • 信息条显示金币余额 — 展示昵称 + 当前竞猜币余额
  • 竞猜玩法根据当前游戏类型动态显示 — 胜负盘、精准总分、精准分差所有游戏均有；元素之王仅冰球大师模式；躺平之王仅拳王。全部需要等待管理员开启竞猜后才会开放
  • 竞猜收益排行榜预览可见 — 按累计竞猜收益（totalBettingWinnings）降序排列，展示排名与段位图标
