🔑 两种登录方式
登录页提供 微信一键登录（面向普通用户）和 管理员登录（面向场馆管理员）两种入口。两种方式登录后获得不同的角色权限。
💬
微信一键登录
角色：user（普通用户）
调用 uni.login 获取微信临时 code，发送到后端 /api/auth/login-wechat 换取 accessToken。用户无感知，无需输入任何信息。
1用户点击「微信登录」按钮
2调用 uni.login({ provider: 'weixin' }) 获取临时 code
3POST /api/auth/login-wechat { code } 换取 token + 用户信息
4本地持久化 token + userInfo，跳转至选择门店页
🛡️
管理员免密登录
角色：admin（管理员）
点击「管理员登录」展开表单，输入 管理员手机号（11位），发送到后端 /api/auth/login-admin。手机号须与后端 ADMIN_PHONE 白名单匹配，否则登录失败。
1点击「管理员登录」按钮，展开手机号输入面板
2输入 11 位手机号（默认预填 ADMIN_PHONE）
3POST /api/auth/login-admin { phone } 获取 token + 管理员信息
4本地持久化 token + userInfo（role=admin），跳转至选择门店页

⚠️

🔑 Token 持久化机制
登录成功后，accessToken 和 userInfo（含 role）分别持久化到微信小程序 Storage。后续请求自动携带 token；页面刷新后自动恢复登录态。
💾
写入时机：登录成功后
setToken → Storage('auth_token')
setUserInfo → Storage('user_info') { id, nickname, phone, role }
🔍
校验时机：每次打开小程序
App.vue → userStore.checkAutoLogin() → fetchProfile()
成功：恢复登录态  |  失败（401）：清除 token + 跳回登录页
🗑️
清除时机：退出登录 / token 失效
userStore.logout() → clearAuth() → 移除 auth_token + user_info
→ reLaunchSafe('/pages/login/index')

🚪 退出登录
退出入口在选择门店页底部的「退出登录」按钮，仅登录状态可见。
1调用 userStore.logout()
2清空内存：token = '' , userInfo = null
3清空 Storage：removeToken() + removeUserInfo()
4reLaunchSafe('/pages/login/index') 跳回登录页
📡 相关 API 接口
接口方法说明
/api/auth/login-wechatPOST微信登录：传入 { code }，返回 { accessToken, expiresIn, user }
/api/auth/login-adminPOST管理员登录：传入 { phone }，返回同上。后端校验手机号白名单
/api/auth/profileGET获取当前用户资料（需 token），失败则触发退出登录
/api/auth/profilePUT更新用户资料，如修改昵称
登录流程设计原则：无感登录 · 角色分离 · 自动恢复 · 安全清除
