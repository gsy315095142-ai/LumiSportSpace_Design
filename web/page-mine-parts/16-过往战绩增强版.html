设计说明

  • 玩法筛选栏 — 顶部水平标签：「全部 / 🏒冰球 / 🥊拳王 / 🤺击剑」，快速切换查看不同玩法
  • 结果标签 — 绿色「胜利」 / 红色「失败」 / 橙色「平局」，替代弹窗版的小字
  • 积分变化行 — 新增！展示 旧积分 → 新积分 + 差值（+N / -N），直观反映实力波动
  • 对阵简化 — 突出「我」的视角：「我(红方) vs 对手」，比弹窗版红蓝双行更简洁
  • 从弹窗升级为独立页面 — 支持滚动分页加载（弹窗版仅 50 条无分页）

💡 对比现有弹窗

现有 JoinMatchHistoryModal 是 70vh 弹窗，信息密度低且无法筛选。升级为独立子页面后可复用 GET /api/player/match-history 接口，后端仅需增加 gameType 筛选参数。
